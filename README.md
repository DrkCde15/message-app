## Menssage-App

## Instalar dependencias
-- npm install express sqlite3 ws cors body-parser(backend)
-- npm install lucide-react(frontend)

## Back-End
**fazer tudo pelo terminal de comando(CMD)**

``mkdir messaging-app``
``cd messaging-app``
``mkdir server``
``cd server``
``npm init -y``

# Compilar arquivo server.js
``node server.js``

## Front-End
``cd ..``
``npx create-react-app client``
``cd client``
``npm install lucide-react``

# Iniciar app
``npm start``
